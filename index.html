What CodeLens Does

A complete visualization toolkit that makes learning data structures and algorithms truly intuitive.

Step-by-Step Execution

Walk through code line by line. Watch variables change, pointers move, and data structures transform in real time.

Live

Stack Memory Visualization

See call stack frames push & pop. Watch local variables, function calls, and return values in a visual stack layout.

Live

Data Structure Visualization

Arrays, matrices, pointers rendered as interactive diagrams. Compare, highlight, and track every element change.

Live

Built-in Terminal Output

See printf & cout output in a macOS-style floating terminal window. Output updates as each step executes.

Adjust speed from 0.5x to 2x. Pause, play, step forward or backward. Slider to jump to any step instantly.
